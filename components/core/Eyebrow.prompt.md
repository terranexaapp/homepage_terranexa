Uppercase wide-tracked kicker that sits above every headline; use `pill` for the hero badge with the glowing dot, `plain` for section labels.

```jsx
<Eyebrow variant="pill">Plataforma agrotech integrada</Eyebrow>
<Eyebrow tone="gold">Gestão que gera resultado</Eyebrow>
```

Tones: `green` (default) and `gold`. Always sentence-or-title-cased input — the component uppercases it.
