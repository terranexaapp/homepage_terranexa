# Home TerraNexa

Página institucional independente criada em HTML, CSS e JavaScript puro.

## Arquivos

- `index.html`: estrutura e conteúdo da página.
- `styles.css`: identidade visual e responsividade.
- `script.js`: menu, dropdowns, galeria, animações e formulário.
- `assets/`: imagens locais fornecidas no pacote de referência.

## Visualizar

Abra `index.html` diretamente ou execute:

```powershell
python -m http.server 4310
```

Depois acesse `http://127.0.0.1:4310`.
